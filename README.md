wgc
===

A simple program for returning a list of words that start with a list of consonants.

For example:

    > wgc lcn

    licentious
    republican
    laplacian
    licensing
    anglican
    holocene
    licensed
    licensee
    licenses
    licensor
    pliocene
    silicone
    laocoon
    license
    pelican
    silicon
    volcano
    falcon
    lacing
    lucian
    lucien
    vulcan

The output is sorted by decreasing length order, using alphabetical order to break ties.
